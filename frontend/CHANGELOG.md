## [0.1.0] - 2024-01-15

### Added
- Initial Next.js 16 setup with App Router
- TypeScript strict mode configuration
- Tailwind CSS 4 integration
- Reusable UI component library (Button, Card, Badge, Input, Textarea, EmptyState)
- Layout components (Navbar, Sidebar, Container, PageHeader)
- Public landing page with hero and features
- Authentication pages (login, register)
- Employer dashboard with sidebar navigation
- Employer pages (dashboard, post job, find workers, rights)
- Worker dashboard with sidebar navigation
- Worker pages (dashboard, browse jobs, rights)
- Centralized API service layer
- TypeScript types for API contracts
- Comprehensive documentation suite
- Environment configuration template
- Professional documentation (README, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY)
- System architecture diagram (Mermaid)

### Technical Details
- Next.js: 16.1.6
- React: 19.2.3
- TypeScript: 5.x
- Tailwind CSS: 4.x
- lucide-react for icons
- Inter font via next/font/google
